Calendar

October 2019

In this project I built a Calendar planner app designed around the concept of modularity. Events on the planner can be further expanded with different event tags, such as locations, notes, lists, media, etc. This allows the user to customize entrees the way they need in an intuitive way.

Feeling Zen or Productive?

Different Calendar styles allow the user to tune the app to their need of inspiring relaxation or full productivity interface.

Customizable UI

The theme of Calendar's interface is fully customizable, allowing the user to set their favourite backgrounds and style.

Core Design Concept Expanded

Every event is fully modular, being at its basic a simple labelled event date. The user can create combinations of tags, such as location and a to do list. Or a longer note with a couple of images. The function allows for high customisability and optimized productivity.
